◆ tempBufferHeap

VkDeviceMemory tempBufferHeap

Optional externally allocated buffer memory for XeSS. If VK_NULL_HANDLE the memory is allocated internally. If provided, the memory must be allocated from memory type that supports allocating buffers. The memory type should be DEVICE_LOCAL. This memory is not accessed by the CPU.

◆ tempTextureHeap

VkDeviceMemory tempTextureHeap

Optional externally allocated texture memory for XeSS. If VK_NULL_HANDLE the memory is allocated internally. If provided, the memory must be allocated from memory type that supports allocating textures. The memory type should be DEVICE_LOCAL. This memory is not accessed by the CPU.
