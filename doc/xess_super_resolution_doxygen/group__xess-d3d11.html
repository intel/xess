◆ xessD3D11Init()

XESS_API xess_result_t xessD3D11Init ( xess_context_handle_t  hContext,
const xess_d3d11_init_params_t *  pInitParams 
)

Initialize XeSS D3D11. This is a blocking call that initializes XeSS and triggers internal resources allocation and JIT for the XeSS kernels. The user must ensure that any pending command lists are completed before re-initialization. When During initialization, XeSS can create staging buffers and copy queues to upload internal data. These will be destroyed at the end of initialization.

Parameters
hContextThe XeSS context handle.
pInitParamsInitialization parameters.
Returns
XeSS return status code.
