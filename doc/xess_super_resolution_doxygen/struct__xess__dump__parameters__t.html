xess_dump_elements_mask_t dump_elements_mask

Bitset showing set of elements that must be dumped. Element will be dumped if it exists and corresponding bit is not 0. Since it's meaningless to call Dump with empty set value of 0 will mean DUMP_ALL_INPUTS
